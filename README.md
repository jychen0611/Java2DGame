# Java2DGame
![image](https://user-images.githubusercontent.com/56495126/131082791-b3581898-f5fd-47a5-b42d-f4fa04a37c73.png)

玩家可以選擇不同的坦克進行對戰
操作方式
P1:
移動:上下左右 射擊:空白鍵
P2:
移動 : WSAD 射擊 : F
血量先歸零者為輸家
吃到血包可以回血
